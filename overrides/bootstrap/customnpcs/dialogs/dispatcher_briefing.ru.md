# Диспетчер якоря — диалог «Брифинг»

## Текст (экран 1)

**[ Сигнал якоря ]**

Сканер зафиксировал пульс в геотермальном разломе — ~160 блоков на северо-восток.
Пар мешает приёмнику. Лагерь развёрнут правильно.

Вы — полевые исследователи протокола **CoopTech**.
Откройте книгу квестов (**Gr**), закрепите цель на HUD.

## Опция «Принять приказ»

- Тип: **Command** (без слэша)
- Команда: `ftbquests change_progress @dp complete T101000000000021`
- Текст кнопки: `§aПринять приказ`

## Опция «Ещё вопросы»

**[ Подсказки ]**

- **JEI** — поиск рецептов по названию.
- **Кооп:** оба игрока в одной FTB Team.
- **Разлом:** координаты на табличке у выхода из лагеря.

## Внешний вид NPC

- Имя: `§dДиспетчер якоря`
- Роль: Follower / не двигать
- Скин: любой «офицер» / teal-cyan плащ
- Позиция: маркер `[CNPC] Диспетчер якоря`
