# CoopTech — лагерь экспедиции (Prologue)

Сид `5998437775939784656` · разлом **122, 90, -102**

## Быстрый старт (5 мин)

1. **Creative + OP** (или сервер с `enable-command-blocks=true`).
2. Телепорт на спавн: `/spawn` или встаньте где хотите лагерь.
3. Встаньте **на землю** в центре будущей площадки.
4. Выполните:

```
/function cooptech:camp/build
```

Построится площадка, костёр, палатки, тропа к разлому, маркеры NPC. `setworldspawn` ставится под вами.

5. **Reload** (если функция не найдена): `/reload` после `python scripts/sync_to_tlauncher.py`.

---

## CustomNPCs (15–20 мин)

1. Палочка CNPC → создать NPC на каждом **цветном маркере**.
2. После создания NPC: `/kill @e[type=armor_stand,tag=cooptech_npc]` (удалить маркеры).
3. Диалоги — копировать из `overrides/bootstrap/customnpcs/dialogs/`.

### Диспетчер (обязательно для квеста)

| Поле | Значение |
|------|----------|
| Диалог | «Брифинг» → опция **Command** |
| Команда | `ftbquests change_progress @dp complete T101000000000021` |
| Текст | см. `dispatcher_briefing.ru.md` |

### Остальные (опционально)

| NPC | Файл |
|-----|------|
| Квартирмейстер | `vendor.ru.md` |
| Наставник | `mentor.ru.md` |
| Техник, патрульный | сидят / патруль 15 блоков, без команд |

### Сервер

- `NpcUseOpCommands=true` в config CustomNPCs (после первого запуска мода).
- `enable-command-blocks=true`
