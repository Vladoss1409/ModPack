## Обзор

**ExtendedAE** (ExAE) — большое QoL-дополнение к [Applied Energistics 2](../appliedenergistics2-forge/): ускоряет крафт, упрощает производство процессоров и добавляет продвинутые шины и терминалы.

## Ключевые блоки и предметы

| Блок / предмет | Назначение |
|----------------|-----------|
| **Circuit Cutter (Слайсер)** | Режет блоки кристаллов/кремния на печатные платы — **без прессов инскрайбера** |
| **Crystal Assembler** | Мгновенно выращивает и чинит чистые кристаллы; вместе со слайсером заменяет инскрайбер |
| **Assembler Matrix** (многоблок) | Масштабируемый крафтер для экстремальной скорости |
| **Extended Molecular Assembler** | Молекулярный сборщик с ×8 параллельным крафтом |
| **Extended Inscriber / Charger** | Параллельное производство плат и зарядка |
| **36-слотовый Pattern Provider / Interface** | Огромные наборы шаблонов в одном блоке |
| **ME Wireless Connector** | Дальний беспроводной мост между частями сети |
| **Precise / Mod / Tag Export & Storage Bus** | Экспорт по слотам, по mod ID или по тегу предметов |
| **Infinity Cells (вода/булыжник)** | Бесконечный запас базовых ресурсов |
| **Extended Pattern Access Terminal** (провод + беспровод) | Удобное управление всеми шаблонами |

## Как использовать

- **Слайсер + Crystal Assembler**: положите блок кристаллов/кремния в Circuit Cutter → получите печатные платы; кристаллы соберите в Crystal Assembler. Эта связка полностью **убирает нужду в прессах и инскрайбере** — огромное ускорение старта AE2.
- **Assembler Matrix / Extended Molecular Assembler**: для массового автокрафта, когда обычные сборщики не успевают.
- **Tag/Mod Storage Bus**: фильтруйте целые категории предметов одним правилом.

## Роль в сборке

Снимает главные «боли» AE2: медленное производство процессоров и ограниченные шины. Один из самых полезных технических аддонов пакета.
