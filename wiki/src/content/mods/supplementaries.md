## Обзор

**Supplementaries** — большой набор **декоративных и функциональных** блоков в ванильном стиле. Десятки мелочей, которые оживляют базу и добавляют полезные механики.

## Примеры блоков и предметов

| Блок / предмет | Назначение |
|----------------|-----------|
| **Sconce / Candle Holder** | Настенные светильники |
| **Urn, Jar, Sack** | Ёмкости для хранения и декора (в банку видно содержимое) |
| **Safe** | Личный сейф, привязанный к владельцу (ключ) |
| **Item Shelf / Pedestal** | Витрины для предметов |
| **Faucet** | Кран для перелива жидкостей между ёмкостями |
| **Hourglass** | Песочные часы — источник настраиваемого редстоун-сигнала |
| **Cannon** | Пушка (стреляет зарядами) |
| **Notice Board / Sign Post** | Доски объявлений и указатели-стрелки |
| **Rope** | Верёвка для спуска/подъёма |
| **Flags, Globe, Statue** и др. | Чисто декоративные детали |

## Механики

- Многие блоки **функциональны**: Faucet + Jar автоматизируют мелкий перелив, Hourglass даёт таймерный редстоун, Sack/Urn — компактное хранение.
- Большинство настраивается в конфиге (можно отключать отдельные предметы).
- Тесно связан с [Amendments](../amendments/) — тот добавляет поведение, а Supplementaries — блоки.

## Роль в сборке

Украшение и «обживание» базы + мелкие удобные механики. Не влияет на тех-прогрессию, но заметно повышает уют и функциональность построек.
