## Обзор

**Flux Networks** — **беспроводная передача энергии (RF/FE)**. Создаёт «облачную» энергосеть: генераторы в одном месте, потребители — в другом (даже в другом измерении), без кабелей между ними.

## Ключевые блоки и предметы

| Блок / предмет | Назначение |
|----------------|-----------|
| **Flux Plug** | **Вход** в сеть: снимает энергию с генератора/аккумулятора |
| **Flux Point** | **Выход** из сети: отдаёт энергию машине/потребителю |
| **Flux Controller** | Ядро большой сети: снимает лимит пропускной способности, хранит энергию |
| **Flux Storage (Herculean/Gargantuan)** | Беспроводные аккумуляторы огромной ёмкости |
| **Flux Configurator** | Инструмент настройки сети и подключения |

## Как использовать

1. Создайте сеть в меню (**Flux Configurator** или интерфейс блока), задайте имя/пароль.
2. Поставьте **Flux Plug** на генератор (он «вливает» энергию в сеть).
3. Поставьте **Flux Point** на потребителя (он «выкачивает» энергию из сети).
4. Все блоки одной сети обмениваются энергией **беспроводно**, независимо от расстояния и измерения.
5. Для больших мощностей добавьте **Flux Controller** — иначе есть лимит на точку.

Настраиваются приоритеты и лимиты по каждому Plug/Point.

## Роль в сборке

Убирает «спагетти» из кабелей: единая энергосеть базы, генерация на удалённой электростанции, потребление где угодно. Идеально для кооператива с распределённой базой. Хорошо сочетается с [Powah](../powah/), [Mekanism](../mekanism/), [Thermal](../thermal-foundation/).
