## Обзор

**Silent Gear** — модульная система **инструментов и брони**: вы сами собираете предмет из **частей** (лезвие, стержень, наконечник, ткань и т.д.) и **материалов**. Характеристики считаются из комбинации — можно сделать уникальное снаряжение под задачу.

## Ключевые механики

| Механика | Описание |
|----------|----------|
| **Модульная сборка** | Инструмент = части + материалы; каждая часть влияет на урон, прочность, скорость |
| **Материалы** | Дерево → камень → металлы → экзотика (из других модов); у каждого свои статы |
| **Трейты (черты)** | Особые свойства материала: огонь, магнит, лёгкость, бронебойность |
| **Синергии** | Некоторые комбинации материалов дают бонусы |
| **Ремонт** | Чинится материалом той же категории, не на наковальне |
| **Зачарования** | Поддерживает ванильные и кастомные |

## Ключевые предметы

| Предмет | Назначение |
|---------|-----------|
| **Blueprint / Template** | Чертёж для сборки инструмента/брони |
| **Material Grader** | Оценивает качество материала (★) — влияет на итоговые статы |
| **Salvager** | Разбирает Silent Gear-предметы на части |
| **Gear Crafting Table** | Верстак для сборки снаряжения |
| **Coating / Upgrade** | Покрытия и улучшения поверх готового предмета |

## Как использовать

1. Соберите **Blueprint** нужного инструмента (меч, кирка, броня).
2. Выберите **материалы** для каждой части (смотрите статы в JEI — урон, прочность, трейты).
3. Соберите на **Gear Crafting Table**.
4. При желании улучшите **Material Grader**'ом и добавьте **Coating**.

## Роль в сборке

Гибкая альтернатива фиксированным наборам ([Mekanism Tools](../mekanismtools/), ваниль). Можно собрать идеальную кирку под добычу или меч под боссов, комбинируя материалы из разных модов.
